# 🎷 GorJazz Website - Project Roadmap

This project follows the **OpenClaw High-Performance Website Standard**. Our goal is to create more than just a landing page; we are building a "Topical Authority" hub that is optimized for both human visitors and AI discovery.

## 🚀 Website Success Checklist

### Phase 1: Core Content & Identity
- [x] **Brand Alignment:** Minimalist Jazz Gallery aesthetic (Warm Paper/Charcoal).
- [x] **Mobile-First Design:** Fully responsive layout using Tailwind CSS.
- [x] **Internal Linking:** Seamless navigation between sections and pages.
- [ ] **Asset Integration:** High-res photography of Inbal & Nadav (placeholder currently).

### Phase 2: SEO & AI Readiness (The "Smart" Layer)
- [x] **Semantic HTML:** Correct H1-H3 hierarchy for search engine clarity.
- [ ] **FAQ Schema:** Implement JSON-LD for "Rich Results" (Questions about booking, repertoire, etc.).
- [x] **Meta Optimization:** Unique titles and descriptions for every page.
- [x] **Canonical Tags:** Prevent duplicate content issues.
- [ ] **Local SEO:** Targeted keywords for "Jazz in Tel Aviv" and "Live Music for Events in Central Israel".

### Phase 3: Trust & Credibility (Social Proof)
- [x] **Instagram Integration:** Live Embeds to show "real-time" activity.
- [ ] **Testimonials Section:** Real quotes from clients and venues.
- [ ] **Media Hub:** High-quality audio snippets and YouTube video embeds.
- [x] **Direct Action:** "Zero-friction" contact via WhatsApp and Email.

### Phase 4: Technical Excellence
- [ ] **Performance:** Image optimization (WebP format) and lazy loading.
- [x] **Tracking:** GTM/GA4 integration (Stubs ready).
- [ ] **Security:** SSL/HTTPS verification and secure form handling.

## 🛠️ Tech Stack
- **Framework:** HTML5, Tailwind CSS
- **Fonts:** Playfair Display (Serif), Assistant (Sans-serif)
- **Deployment:** GitHub Pages / Custom Domain
